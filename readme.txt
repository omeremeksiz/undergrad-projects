University Projects Repository

This repository contains projects that I completed during my university courses.

Project 1: Distance Detector

Description: Embedded system project using C/C++, STM32CubeIDE, NUCLEO-L073RZ, and HC-SR04 Ultrasonic Sensor. Outputs are generated via servo motors and LEDs based on detected distances.
Completion Date: June 2022
Project 2: Image Encryption

Description: Computer program developed in C for pixel-level image encryption and decryption. Implements various algorithms to encrypt and decrypt images, ensuring secure access.
Completion Date: January 2022
Project 3: Student Grading System

Description: MATLAB-based program for calculating final letter grades and dynamically visualizing student data. Incorporates class curve considerations and provides interactive charts.
Completion Date: January 2021
Project 4: Communication Engineering Projects

Description: MATLAB-based implementation of communication engineering concepts, including signal processing, modulation, and demodulation techniques. Covers various signal processing scenarios.
Completion Date: June 2023
Project 5: Introduction to Digital Signal Processing Projects

Description: Repository containing projects from the Introduction to Digital Signal Processing course. Demonstrates practical applications of DSP concepts using audio signals.
Completion Date: June 2023
